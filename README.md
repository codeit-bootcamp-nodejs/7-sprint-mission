## [NB]스프린트 미션 2

이 프로젝트는 **JavaScript(ESM)** 기반으로 작성된 간단한 API 테스트용 애플리케이션입니다.
`ProductService`와 `ArticleService`를 통해 원격 API 서버와 통신하며, 상품 및 게시글의 생성/조회 기능을 테스트합니다.

## 🚀 1. 설치(Installation)

### 패키지 설치

```bash
npm install
```

또는

```bash
npm i
```


## 2. 실행 방법(Run App)

```bash
node main.js
```

실행 시 자동으로 아래 테스트가 진행됩니다:

* Product API 테스트

  * 상품 생성
  * 상품 조회
  * 상품 리스트 조회

* Article API 테스트

  * 게시글 생성
  * 게시글 조회
  * 게시글 리스트 조회
