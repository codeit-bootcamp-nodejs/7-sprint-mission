# 🐼 판다마켓 (Panda Market) - 백엔드 데이터베이스 모델링

중고 거래와 커뮤니티 기능을 제공하는 '판다마켓' 서비스의 백엔드 데이터베이스 구조 설계 및 쿼리 구현 프로젝트입니다.

## 🚀 프로젝트 개요
사용자가 상품을 등록하고 거래하며, 자유 게시판을 통해 소통할 수 있는 플랫폼입니다. 데이터의 안전성과 무결성을 고려하여 각 도메인(유저, 상품, 게시글, 댓글, 이미지, 태그) 간의 관계를 설계했습니다.

## ✅ 미션 목표
- [x] **SQL 스키마 설계하기**
  - 유저(비밀번호 8자 이상), 상품(상품명 10자 이내/소개 10자 이상/가격 숫자), 태그(5자 이내) 등 세부 제약 조건 반영
  - 댓글 및 이미지 테이블의 분리 설계를 통한 데이터 무결성 확보
- [x] **ER Diagram 그리기**
  - 서비스 도메인 간의 유기적인 관계 시각화 및 1:N, N:M 관계 정의
- [x] **SQL 쿼리 작성하기**
  - 정보 업데이트, 페이지네이션 기반 조회, 좋아요 개수 집계 등 핵심 비즈니스 로직 14종 구현

## 🛠 기술 스택
- **Database**: PostgreSQL / MySQL (Standard SQL)
- **Modeling**: dbdiagram.io

## 📝 파일 구성
- `schema.sql`: 데이터베이스 테이블 정의 및 제약 조건 설정
- `queries.sql`: 서비스 기능 구현을 위한 핵심 SQL 쿼리문
- `erd.png`: 데이터베이스 개체 관계도 이미지
